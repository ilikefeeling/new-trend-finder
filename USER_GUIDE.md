# 🚀 Next Shorts 사용 설명서

**Next Shorts**는 AI 기반 유튜브 쇼츠 분석 도구입니다. 최신 트렌드를 파악하고, 성공한 영상의 비결을 역설계하여 나만의 바이럴 아이디어를 얻으세요.

---

### 1. 🔍 Trend Scout (트렌드 분석)

현재 유튜브에서 가장 뜨거운 쇼츠의 흐름을 분석합니다.

* **탭 선택**: 상단의 `Trend Scout` 탭을 클릭합니다.
* **지역 설정**: `Korea 🇰🇷` (한국 인기 쇼츠) 또는 `Global 🌐` (글로벌/미국 인기 쇼츠) 중 원하는 지역을 선택합니다.
* **분석 시작**: `Analyze` 버튼을 누르면 AI가 현재 가장 인기 있는 영상을 분석합니다.
* **결과 활용**:
  * **Source Trend**: 원본 영상의 시각적/청각적 성공 패턴 요약.
  * **AI Idea Recommendations**: 내 채널에 바로 적용 가능한 3단계 스크립트 가이드.
  * **Localization Strategy**: (글로벌 선택 시) 해외 트렌드를 한국 시장에 맞게 변형하는 전략.

---

### 2. 📈 Outlier (아웃라이어 역설계)

구독자 수보다 조회수가 훨씬 높은 '떡상' 영상의 성공 비결을 찾아냅니다.

* **탭 선택**: 상단의 `Outlier` 탭을 클릭합니다.
* **키워드 입력**: 분석하고 싶은 주제(예: '요리', '브이로그', '운동')를 입력합니다.
* **검색 실행**: `Run` 버튼을 누릅니다.
* **분석 결과**:
  * **Outlier List**: 구독자 대비 조회수 비율(x8.0배 등)이 높은 영상 5개를 찾아냅니다.
  * **Reverse-Engineering**: AI가 해당 영상들의 제목, 훅(Hook), 시청 지속 전략을 분석하여 '승리 공식'을 요약해 줍니다.

---

### 3. 🛠 서버 실행 및 접속 방법

프로젝트 폴더에서 다음 단계를 따르세요.

1. **터미널 실행**: VS Code 하단 터미널 또는 CMD를 엽니다.
2. **명령어 입력**:

    ```bash
    npm run dev
    ```

3. **접속**: 브라우저 주소창에 `http://localhost:3000`을 입력합니다.

---

> [!TIP]
> **성공 팁**: 글로벌 트렌드에서 발견한 비언어적(Non-verbal) 요소들을 한국어 자막과 함께 구성하면 국내외 시청자를 동시에 잡을 수 있는 영상을 제작할 수 있습니다.
